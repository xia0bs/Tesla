Model S 与 Model 3

Model S 是我们的旗舰款高级轿车，行驶里程更长、加速更快，拥有两个显示屏和更多的个性化配置。在安全性能方面，Model S 有口皆碑；同时，新车主通过车主引荐选购 Model S，还可享受不限里程的免费超级充电服务。

Model 3 是一款更紧凑、更简洁、拥有门槛更低的电动车。虽然 Model 3 是我们推出的最新车型，但它不是 "第三版" ，也并非最豪华、最先进的 Tesla 车型。与 Model S 相同，Model 3 将会拥有同级车中极为杰出的安全性能。

Model 3 规格

安全性

  • 自动紧急制动和碰撞预警
  • 8个安全气囊
  • 电子稳定和牵引力控制系统

便利功能

  • 15 英寸触摸屏，车载地图和导航

  • 语音控制系统
  • 60/40 分离式后排座椅

标准内饰
